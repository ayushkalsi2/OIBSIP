## 🚀 Getting Started
1. Clone or download this repository.
2. Open `index.html` directly in your browser — no build step needed.

To deploy:
- **GitHub Pages**: push to a repo, then enable Pages in repo settings (Settings → Pages → Deploy from branch).
- **Netlify/Vercel**: drag and drop the folder or connect the repo for automatic deploys.

## ✏️ Customization
- Update personal details (name, email, bio) directly in `index.html`.
- Add real project links in the `projects` section.
- Update the `education` section with your actual school/program and dates.
- Swap the `AK` initials avatar for a real photo if desired.
